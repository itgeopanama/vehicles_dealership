Vehicles Dealership Business

This is the module for managing the sales and purchase of vehicles for dealership business

We have Option to add the Dealer Unique Identification Number with company details.

Go to Menu : Settings => Users & Companies => Companies.

Go to Menu : Fleet => Vehicles => Vehicles Registration.

When we are fill the vehicle information and create vehicle then product will create automatically with unique identification number of vehicle.

Go to Menu : Sales => Catalog => Product Variants.

We can see the product is create automatically with unique identification number of vehicle.

Go to Menu : Sales => Orders => Quotations .

We can create sale orders for vehicle product, track orders and manage inventory.

Go to Menu : Purchases => Purchase => Requests for Quotation.

We can create purchase orders for vehicle products.

We can manage the vehicle product inventory.
